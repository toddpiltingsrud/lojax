controlling navigation history

lojax uses the URL's hash to track navigation history. To place an item into the browser's navigation history, preface the URL with "#". 

<a href="#navigationhistory.html" data-method="ajax-get">navigation history</a>

lojax handles the "hashchange" event and queries the page for an element with a data-action or href attribute that matches the hash and has a data-method of "ajax-get". This is how lojax decides whether or not it's a hash that it should handle. Filtering for an "ajax-get" ensures that lojax doesn't issue posts or deletes against the server from users clicking the browser's navigation buttons or clicking a link in an email.
